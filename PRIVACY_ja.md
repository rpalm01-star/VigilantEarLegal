# Vigilant Ear プライバシーポリシー 👂🛰️

**発効日：** 2026年6月7日

## はじめに

Vigilant Ear（以下「当社」）は、お客様のプライバシー保護に取り組んでいます。このプライバシーポリシーでは、アプリが処理する情報の内容、お客様のデバイス上に保持される情報、および特定の機能を提供するためにインターネット経由で送信される可能性のある限定的なデータについて説明します。

## プライバシーの概要

- **コアの音響検知はお客様のデバイス上で動作します。** 音声分類、方向追跡、およびアラートロジックは、お客様のスマートフォンのマイクとセンサーを使用してローカルで動作するよう設計されています。
- **当社はお客様のデータを販売しません**。広告SDKや行動分析SDKも使用しません。
- **当社は音声録音を保存またはアップロードしません。** マイクの音声は検知のためにリアルタイムで処理されます。Vigilant Ear によって音声ファイルとして保存されることはありません。
- **一部の機能はインターネットを使用します** — マップ、悪天候フィード、オプションの音楽識別、道路データ、アプリ内購入、および（iOS の）最小限の診断ログ。これらについては以下で説明します。
- **お客様がコントロールします。** Shazam 音楽識別の無効化、アラートカテゴリのオフ、システム設定での権限の取り消し、またはバックグラウンドリスニングの停止をいつでも行うことができます。

## デバイス上で処理される情報

お客様の許可に基づき、Vigilant Ear は以下の情報に**ローカルで**アクセスします：

- **マイク音声** — 環境音（サイレン、車両、ドアベル、近くにいる人など）のリアルタイム検知とアラートのトリガーに使用します。
- **位置情報** — 検知された音と気象警告エリアをマップ上に配置し、方向案内を改善するために使用します。
- **デバイスの向きと動き** — 方位精度の向上に使用します。

このデバイス上処理がアプリの核心です。競合アプリの多くは、分析と収益化のために音声をクラウドにストリーミングしています。Vigilant Ear は異なる設計思想のもとに作られています：お客様の音響認識パイプラインはスマートフォン自体で動作するよう設計されています。

## ネットワークとサードパーティサービス

特定の機能を使用する場合、またはアプリの動作に必要な場合に、**限定的なデータがデバイスの外に送信され**、各サービス固有のプライバシーポリシーのもとでサードパーティによって処理されることがあります：

*   **マップ表示**
    *   *送信される情報：* マップタイルのリクエスト、マップのレンダリングに必要なビューポートとおおよその位置情報
    *   *プロバイダー：* Apple Maps / MapKit
*   **悪天候アラート**
    *   *送信される情報：* 公開の CAP/Atom 気象フィードへのリクエスト。フィードの選択とデバイスの位置情報からお客様のおおよその地域が推測される場合があります
    *   *プロバイダー：* 米国国立気象局、MeteoGate、WMO、および関連する公共アラートソース
*   **音楽識別（オプション）**
    *   *送信される情報：* 音楽が検知され Shazam が有効な場合に短い音声フィンガープリント（設定でオフにできます）
    *   *プロバイダー：* Apple Shazam / ShazamKit
*   **道路コンテキスト**
    *   *送信される情報：* お客様の位置周辺のマップセクターに基づく匿名の Overpass API クエリ
    *   *プロバイダー：* Overpass API 経由の OpenStreetMap 貢献者
*   **サブスクリプションと購入**
    *   *送信される情報：* 購入トークンとサブスクリプションのステータス
    *   *プロバイダー：* Apple App Store
*   **アプリ内法的ドキュメント**
    *   *送信される情報：* アプリ内でプライバシーポリシー、利用規約、またはサポートページを開いたときの標準的なWebリクエスト
    *   *プロバイダー：* GitHub Pages（ドキュメントホスティング）

当社はマップ、気象、音楽ラベル、購入機能を提供するためにこれらのサービスを選択しています。**Wingdings は、これらのプロバイダーからお客様のマイク音声、位置情報の履歴、または連絡先情報を受け取りません。**

## Wingdings が収集する情報

### リモートテレメトリまたは診断なし

Vigilant Ear はお客様のデバイス上で完全にローカルに動作するよう設計されています。当社はリモートテレメトリ、クラッシュログ、診断記録、または使用状況の分析情報を当社のサーバーで収集・送信・保存しません。すべての処理はお客様のハードウェアにローカルに保持されます。

## 当社が行わないこと

当社は以下のことを**行いません**：

- お客様の個人情報の販売またはレンタル
- 当社のサーバーへの環境音声録音の保存
- 広告ネットワーク、クロスアプリトラッカー、または行動プロファイリング SDK の実行
- Wingdings への継続的な位置情報トレイルのアップロード

## お客様の選択とコントロール

お客様は以下のことができます：

- iOS 設定で**権限を取り消す**（マイク、位置情報、通知）
- アプリ設定で **Shazam 音楽識別を無効にする**
- **個別のアラートカテゴリをオフにする**（サイレン、気象、ドアベルなど）
- すべてのアラートカテゴリを無効にして**バックグラウンドリスニングを停止する**
- カスタマイズ内の**Sample Alerts デモ**を使用して、実際の緊急事態を示さずにローカルで通知をプレビューする
