# Vigilant Ear のプライバシーポリシー 👂🛰️

**発効日:** 2026年7月11日

## はじめに

Vigilant Ear (以下「当社」または「私たちの」) は、あなたのプライバシーを保護することをお約束します。このプライバシーポリシーでは、アプリがどのような情報を処理し、何がデバイス上に保持され、特定の機能を提供するために限られたデータがインターネット経由で送信されるのはどのような場合かを説明します。

## プライバシーの概要

- **コアの音響検出はデバイス上で実行されます。** 音の分類、方向の追跡、ライブキャプション、およびアラートロジックは、お使いの電話のマイクとセンサーを使用してローカルで機能するように設計されています。
- **当社はあなたのデータを販売しません**。また、広告や行動分析 SDK を使用しません。
- **音声録音の保存やアップロードは行いません。** マイクの音声は、検出と (有効になっている場合) キャプションのためにリアルタイムで処理されます。後で再生したりクラウド分析したりするために、Vigilant Ear によって音声ファイルとして保存されることはありません。
- **一部の機能はインターネットを使用します** — マップ、悪天候フィード、オプションの音楽識別、道路データ、アプリストアでの購入、*あなたの*デバイス間のオプションのマルチ電話メッシュトラフィック、およびアプリ内の法的ページの読み込み。これらについては以下で説明します。
- **コントロールはあなたにあります。** Shazam の音楽識別を無効にする、アラートカテゴリをオフにする、Constellation をオフのままにする、システム設定で権限を取り消す、またはバックグラウンドでのリスニングをいつでも停止することができます。

## デバイス上で処理される情報

あなたの許可を得て、Vigilant Ear は**ローカルで**以下にアクセスします：

- **マイクの音声** — 環境音 (サイレン、車両、ドアベル、赤ちゃんの泣き声、近くにいる人々など) を検出し、方向を推定し、(スピーカーモードがオンの場合) ライブキャプションとオプションのデバイス上翻訳を生成するためにリアルタイムで使用されます。
- **音声認識 (デバイス上)** — キャプションが有効になっている場合、デバイスの音声フレームワークが近くの音声を電話上のテキストに文字起こしします。キャプションのテキストはライブで表示され、永久的な文字起こし履歴として Vigilant Ear によってアーカイブされることはありません。デバッグログにはキャプションの内容は含まれません。
- **位置情報** — 検出された音や気象アラートエリアをマップ上に配置し、方向のガイダンスを改善するために使用されます。
- **デバイスの向きとモーション** — 方位の精度を向上させるために使用されます。
- **カメラ (オプション)** — カメラ AR の「音を見る」ビューを開いた場合にのみ使用され、ライブカメラプレビューにマーカーをピン留めできます。カメラフレームはデバイス上での表示に使用され、音声認識のために Vigilant Ear によってアップロードされることはありません。
- **Apple Watch (オプション)** — Watch コンパニオンが利用可能な場合、手首で一目で確認できるように、アラートラベルと方向の手がかりがペアリングされた Watch にリレーされる場合があります。

このデバイス上での処理がアプリの心臓部です。競合アプリは分析と収益化のために音声をクラウドにストリーミングすることがよくあります。Vigilant Ear は異なる方法で構築されています。あなたの音響認識パイプラインは、電話自体で実行されるように設計されています。

## ネットワークおよびサードパーティのサービス

特定の機能を使用する場合 — またはアプリが機能するためにそれらを必要とする場合 — **限られたデータがデバイスから離れ**、各々のプライバシーポリシーの下でサードパーティのサービスによって処理される場合があります：

*   **マップの表示**
    *   *送信されるもの:* マップタイルのリクエスト。マップをレンダリングするために必要な場合のマップのビューポートと概算の位置情報
    *   *プロバイダー:* Apple Maps / MapKit
*   **悪天候アラート**
    *   *送信されるもの:* 公共の CAP/Atom 天気フィードへのリクエスト。フィードの選択とデバイスの場所から一般的な地域が推測される場合があります
    *   *プロバイダー:* 米国国立気象局 (NWS)、MeteoGate (ヨーロッパ)、中国気象局 (CMA)、韓国気象庁 (KMA)、WMO 関連の公開ソース、および同様の公開アラートフィード
*   **地震アラート**
    *   *送信されるもの:* 世界共通の公開地震サマリーフィード 1 件へのリクエスト — リクエストには位置情報や地域情報は一切含まれません。報告された地震がお近くかどうかの判定には、デバイスの位置情報がデバイス上でのみ使用されます
    *   *プロバイダー:* 米国地質調査所 (USGS) の公開地震フィード
*   **音楽識別 (オプション, Power Pack+)**
    *   *送信されるもの:* 音楽が検出され、Shazam が有効になっている場合の短いオーディオフィンガープリント — 生音声ではありません (設定でオフにできます)
    *   *プロバイダー:* Apple Shazam / ShazamKit
*   **道路のコンテキスト**
    *   *送信されるもの:* あなたの現在地周辺のマップセクターに基づく匿名化された Overpass API クエリ
    *   *プロバイダー:* Overpass API を介した OpenStreetMap コントリビューター
*   **購入とエンタイトルメント**
    *   *送信されるもの:* オプションの1回限りの Power Pack+ ロック解除 (サブスクリプションではありません) のための購入トークンとエンタイトルメント / トライアルのステータス
    *   *プロバイダー:* Apple App Store
*   **Constellation メッシュ (オプション, Power Pack+)**
    *   *送信されるもの:* マルチ電話の Constellation を有効にすると、参加しているデバイスは、共有された状況把握に必要な音響メタデータを交換します — 例えば、利用可能な場合の相対姿勢 / Ultra-Wideband の測距、方位、音のラベル、一時的なキャプションテキストなど。トラフィックはあなたがリンクした電話間のピアツーピアです。Wingdings はこのオーディオパイプラインのためのクラウドメッシュリレーを運営していません。
    *   *プロバイダー:* あなたのデバイス間の Apple フレームワーク (例: Network / Nearby Interaction)
*   **アプリ内の法的文書**
    *   *送信されるもの:* アプリ内でプライバシーポリシー、利用規約、サポート、または製品の README ページを開いた場合の標準的なウェブリクエスト
    *   *プロバイダー:* GitHub (ドキュメントホスティング)

私たちは、マップ、天気、音楽ラベル、購入、およびマルチデバイス機能を提供するためにこれらのサービスを選択しています。**Wingdings は、これらのプロバイダーからあなたのマイク音声、継続的な位置情報履歴、または連絡先情報を受け取ることはありません。**

## Wingdings が収集するもの

### リモートでの遠隔測定 (テレメトリ) や診断はありません

Vigilant Ear は、お使いのデバイス上で完全にローカルで動作するように設計されています。当社は、Wingdings のサーバーにリモートテレメトリ、クラッシュログ、診断記録、または使用状況分析を収集、送信、または保存することはありません。トラブルシューティングのために、オプションの**ローカル**デバッグログがデバイス上に書き込まれる場合がありますが、これらはテレメトリパイプラインとしてアプリによってアップロードされることはなく、キャプションテキストはエクスポートされたデバッグコンテンツに含まれません。

## 当社がしないこと

当社は以下を**行いません**：

- あなたの個人情報を販売または貸与すること
- 環境音の録音を当社のサーバーに保存すること
- 広告ネットワーク、クロスアプリトラッカー、または行動プロファイリング SDK を実行すること
- あなたの継続的な移動軌跡を Wingdings にアップロードすること
- クラウドの音声または音声認識のために生のマイク音声をアップロードすること

## あなたの選択とコントロール

あなたは以下のことができます：

- iOS の設定で**権限を取り消す** (マイク、位置情報、カメラ、通知、音声認識)
- Power Pack+ / 環境設定で **Shazam 音楽識別を無効にする**
- **個々のアラートカテゴリをオフにする** (サイレン、天気、ドアベル、赤ちゃんなど)
- すべてのアラートカテゴリが無効になっている場合に**バックグラウンドでのリスニングを停止する**
- **Constellation をオフのままにする**ことで、他の電話とメッシュメタデータが共有されないようにする
- **機能プレイグラウンドを使用する**ことで、実際の緊急事態をほのめかすことなく、明確な PREVIEW ウォーターマークを使用してローカルでアラートと機能をプレビューする
